# Banking System Mini-Project
This Mini Project is a Banking System Case Study that uses Python OOP

## Installation
Update db_details.py with MySQL HOST, USER and PW  

Execute  
```bash
./db_setup.py
```

## Usage
```bash
./welcome.py
```
